# 카카오톡 링크 공유 이미지 캐시 문제 해결 가이드

## 문제
카카오톡으로 링크를 공유할 때 썸네일 이미지가 업데이트되지 않는 경우가 있습니다. 이는 카카오톡의 캐시 때문입니다.

## 자동 해결 (이미 구현됨)
- 이미지 URL에 버전 쿼리 파라미터가 자동으로 추가됩니다
- 이미지가 업데이트되면 `updated_at` 기반으로 버전이 변경되어 카카오톡이 새로운 이미지로 인식합니다
- 예: `https://ejdc.eungming.com/uploads/images/main_cover.jpg?v=1735632000000`

## 수동 캐시 초기화 (필요시)

### 카카오톡 캐시 초기화 방법

1. **카카오 디벨로퍼스 공유 디버거 접속**
   - https://developers.kakao.com/tool/clear/og

2. **카카오 계정으로 로그인**

3. **URL 입력**
   - `https://ejdc.eungming.com` 입력
   - ⚠️ 주의: `http`와 `https`를 구분하여 입력 (SSL 인증서가 있다면 `https` 사용)

4. **디버그 버튼 클릭**

5. **캐시 초기화 버튼 클릭**
   - 오픈그래프 정보가 초기화됩니다
   - 이제 카카오톡에서 새로운 이미지가 표시됩니다

### 텔레그램 캐시 초기화 방법

1. 텔레그램에서 "웹페이지 봇" 검색
2. '시작' 클릭 (또는 `/start` 입력)
3. 업데이트할 페이지 URL 입력: `https://ejdc.eungming.com`
4. 링크 미리보기가 업데이트됩니다

## 참고
- 이미지를 업로드/변경하면 자동으로 버전이 변경되어 대부분의 경우 자동으로 해결됩니다
- 여전히 이전 이미지가 표시되는 경우에만 수동 캐시 초기화를 진행하세요
